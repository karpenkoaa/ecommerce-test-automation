# E-commerce Test Automation Framework

This project is a test automation framework built using Python, Selenium, and Behave (BDD).

## Overview
The framework is designed to automate end-to-end testing of a web-based e-commerce application, covering key user flows and ensuring application reliability.

## Features
- Automated UI testing using Selenium WebDriver
- BDD approach with Behave and Gherkin syntax
- Page Object Model (POM) for maintainable and scalable tests
- Modular and reusable test components

## Test Scenarios Covered
- User login
- Product search
- Add to cart
- Checkout flow

## Tech Stack
- Python
- Selenium WebDriver
- Behave (BDD)

## Project Structure
features/    # BDD feature files
pages/       # Page Object Model classes
app/         # Supporting application logic

## How to Run
pip install -r requirements.txt
behave
